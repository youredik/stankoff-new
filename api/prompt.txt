Загрузка фото и видеофайлов


На странице заявки, между формой для смены статуса и секцией Активность нужно добавить возможность загружать фото и видео файлы.
В качестве хранилища нужно использовать Yandex Object Storage - документация к нему https://yandex.cloud/ru/docs/storage/?from=int-console-empty-state
Нужно использовать AWS SDK для PHP - https://yandex.cloud/ru/docs/storage/tools/aws-sdk-php
Бакет имеет название stankoff, а файлы в нем нужно хранить  в папке media/<id заявки>
Нужно чтобы работало Drag and drop, без подтверждения для загрузки файлов. То есть перетаскиваешь файл в нужную область и файл автоматически загружается.
Также нужно сделать возможность просмотра фото и видео, а так же скачивания.
Также нужно сделать возможность удаления файлов.
Примени лучшие практики разработки и также UX UI, чтобы было все понятно и удобно.
